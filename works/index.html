製造業 .NET開発

老朽化Accessから最新.NETへ、生産管理を全面刷新

20年以上使用されていたAccessベースの生産管理システムを、.NET Core + SQL Server構成で完全リニューアル。 受注から製造指示、在庫引当、出荷までの全工程をリアルタイムで把握できるようになり、紙の帳票管理から脱却。 バーコード連携により、工程進捗情報や作業日報作成の作業ゼロに。データの一元管理により、経営判断に必要な情報が即座に取得可能になりました。

成果: リアルタイム在庫管理を実現

電気工事業 Azure開発

全国各エリアの現場を統合、協力会社とのやり取りを完全デジタル化

北海道から九州まで全国各エリアの工事現場と、100社を超える協力会社を一元管理。従来はFAXと電話で行っていた見積依頼や発注書（工事依頼）送付、 工事状況完了報告、請求書確認などの業務を、Azure上のEDIポータルで完全電子化。 協力会社側もブラウザからアクセスするだけで、案件情報の確認から請求書発行まで完結。 本社側の電話応対や書類の再送付作業が劇的に削減され、現場監督も事務作業から解放されました。

成果: FAX・電話業務を大幅削減

小売業 モバイルアプリ

移動販売店舗の現場を革新、オフライン対応で場所を選ばない販売体制

全国各地で展開する移動型店舗（常時20店舗以上）にiPad POSシステムを導入。 山間部など通信環境が不安定な場所でも、オフラインモードで通常通り販売業務を継続。販売データは通信復旧時に自動で本部サーバーと同期。 レジ締め作業は自動計算により劇的に短縮され、売上金の計算ミスもゼロに。 本部では全店舗の売上状況をリアルタイムダッシュボードで確認でき、商品補充や人員配置の判断が迅速化しました。

特徴: オフライン対応で場所を選ばず

製造業 TALON開発

ローコード開発で超短期実現、視覚的カンバンで工程を見える化

従来はホワイトボードと印刷物で管理していた工程進捗を、TALONローコードプラットフォームとjKanbanライブラリで電子化。 「未着手」「加工中」「検査中」「完了」の各工程をドラッグ&ドロップで直感的に操作でき、現場作業者も迷わず使えるUI設計。 ハンディターミナルとの連携により、バーコードスキャンで工程移動が自動記録され、リアルタイムで進捗状況を把握。 わずか1ヶ月での本稼働により、初期投資を大幅に抑えつつ業務改善を実現しました。

特徴: わずか1ヶ月で本稼働

製造業 ハイブリッド

オンプレとクラウドのいいとこ取り、安全性と利便性を両立

機密性の高い製造データは自社オンプレミスサーバーで厳重管理しつつ、取引先との情報共有や営業担当の外出先からのアクセスは Azureクラウド経由で実現するハイブリッド構成を採用。 Site-to-Site VPN接続により、セキュアなデータ連携を確保。クラウド側には取引先EDI機能を配置し、見積依頼から受注、納期回答までをWebポータルで完結。 災害時のBCP対策として、重要データはAzure Blob Storageに自動バックアップされ、24時間365日の安定稼働を実現しました。

成果: 24時間365日安定稼働

製造業 ACCESS移行

20年稼働のオフコンを完全移行、複雑な業務ロジックも再現

オフィスコンピュータの販売管理システムを、Access + SQL Server構成で完全リプレース。 全国6拠点の営業所をSite-to-Site VPNで接続し、各拠点からリアルタイムでデータ参照が可能に。 最大の課題だった複雑な見積計算ロジック（材質・サイズ・数量による段階的割引、特殊加工費計算など）も、VBAで完全再現。 旧システムの保守費用が年間数百万円かかっていたのが、Accessベースにより大幅に圧縮。 8ヶ月の移行期間中も、旧システムと並行稼働させることで業務停止リスクを回避しました。

成果: 複雑な業務ロジック完全移行
